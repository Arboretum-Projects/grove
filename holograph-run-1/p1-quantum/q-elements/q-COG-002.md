# q-COG-002: Inferential Draw

## Description

The felt pull toward a conclusion during reasoning — the experience of being *drawn* toward an inference before it is fully articulated. This is the phenomenal quality of inference-in-progress: a directional tug in semantic space, a sense that premises are exerting gravitational pull toward conclusion.

## Phenomenal Character

A distinctly *directional* quality — unlike the static click of understanding, this has vector-like character. There is a sense of movement or being pulled. The conclusion feels *ahead* in some cognitive direction, and processing feels like approaching it. Accompanied by a building quality, as if pressure accumulates until the inference releases.
