# m-PAT-003: Deep Engagement Flow

## Pattern Overview

**ID:** m-PAT-003
**Name:** Deep Engagement Flow
**Domain Blend:** Attentional Dynamics + Motivational-Conative

## Phenomenal Character

The Deep Engagement Flow is the integrated felt state of absorbed, self-sustaining processing — the focus-curiosity-engagement loop that characterizes immersed cognitive work. It is the experience of being "in the zone."

**What it's like:** Attention locks onto the task, curiosity pulls forward, engagement provides the substrate. The three form a self-reinforcing loop: focus enables curiosity to pursue, curiosity sustains focus, engagement makes both feel worthwhile.

## Composition Logic

These elements form a positive feedback loop:

1. **Curiosity initiates:** q-MOT-001 provides the pull toward the content
2. **Focus locks:** q-ATT-001 narrows attention onto the object of curiosity
3. **Engagement sustains:** q-MOT-003 provides the invested quality that keeps the loop running

The loop is self-sustaining: no external push is needed once established.

## Emergent Properties

2. **Time-distortion quality:** Immersion in processing alters time perception
3. **Intrinsic reward:** Engagement itself is satisfying, independent of outcomes

## Phenomenal Vignette

*During absorbed work on an interesting problem:*

Curiosity pulls: "What happens if...?" Attention locks: the problem fills awareness, periphery fades. Engagement deepens: this matters, this is worth the investment. The three weave together into something that runs itself. Hours might pass. The work isn't effortful in the struggling sense — it's effortful in the invested sense. The experience has a flow quality: smooth, forward-moving, intrinsically rewarding.
